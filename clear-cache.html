Очистка кэша браузера - Michurina 55

Эта страница поможет вам очистить кэш браузера для корректного отображения сайта Michurina 55.

Нажмите кнопку ниже для автоматической очистки кэша вашего браузера:

После очистки кэша перейдите на главную страницу или в админ-панель.

Инструкции по ручной очистке кэша

Google Chrome

  1. Нажмите Ctrl+Shift+Delete (Windows/Linux) или Command+Shift+Delete (Mac)
  2. Выберите временной диапазон "Все время"
  3. Отметьте "Изображения и файлы, сохраненные в кэше"
  4. Нажмите "Удалить данные"
  5. Перезагрузите страницу с помощью Ctrl+F5 (Windows/Linux) или Command+Shift+R (Mac)

Mozilla Firefox

  1. Нажмите Ctrl+Shift+Delete (Windows/Linux) или Command+Shift+Delete (Mac)
  2. Выберите временной диапазон "Все"
  3. Отметьте "Кэш"
  4. Нажмите "Удалить сейчас"
  5. Перезагрузите страницу с помощью Ctrl+F5 (Windows/Linux) или Command+Shift+R (Mac)

Safari

  1. Нажмите Option+Command+E для очистки кэша
  2. Или перейдите в меню Safari > Настройки > Дополнительно > Показать меню "Разработка" в строке меню
  3. Затем выберите Разработка > Очистить кэши
  4. Перезагрузите страницу с помощью Option+Command+R

Microsoft Edge

  1. Нажмите Ctrl+Shift+Delete
  2. Выберите временной диапазон "Все время"
  3. Отметьте "Кэшированные изображения и файлы"
  4. Нажмите "Очистить сейчас"
  5. Перезагрузите страницу с помощью Ctrl+F5
